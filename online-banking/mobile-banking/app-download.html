    How to set up the app

    1. Download the app

    Download the app from the App Store or Google Play. Then open it once it’s ready.

    2. Log on or register

    If you already use Internet Banking, select Log on. If not, select Register and follow our registration guide.

    3. Log on details

    We’ll ask for your User ID and password. Use the same details you do for Internet Banking.

    4. Memorable information

    Next we’ll need 3 characters of your memorable information. The number above each box tells you which to enter.

    5. Security check

    Now we’ll do a quick security check to make sure it’s you. Choose a number for us to call you on.

    6. Confirm it's you

    You’ll get a 4-digit code on your screen. Then we’ll call you on the number you chose. You’ll be asked to read out the code or key it in using the numbers on your phone.

    7. All done

    That’s it, you’re up and running with the app.
